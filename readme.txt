== Description ==

Плагин Kash Back автоматически добавляет ID текущего пользователя к внешним партнерским ссылкам WooCommerce.
Это позволяет отслеживать, какие пользователи переходят по партнерским ссылкам и совершают покупки.

Основные возможности:
* Автоматическое добавление параметра user_id к внешним партнерским ссылкам
* Совместимость со стандартными функциями WooCommerce
* Поддержка локализации

== Installation ==

1. Загрузите плагин через интерфейс WordPress или загрузите ZIP-архив
2. Активируйте плагин через меню "Плагины"
3. Плагин начнет работу автоматически

== Frequently Asked Questions ==

= Работает ли плагин с любыми внешними партнерскими ссылками? =

Да, плагин работает со всеми внешними партнерскими товарами WooCommerce.

= Безопасно ли использование этого плагина? =

Да, плагин использует безопасные методы WordPress для модификации URL и не изменяет базу данных.
